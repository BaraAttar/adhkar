# adhkar

هذا مشروع تطبيق Flutter عربي يهدف إلى مساعدة المستخدم على قراءة الأذكار اليومية بشكل بسيط ومنظم.

## فكرة المشروع

التطبيق يعرض مجموعة من الأذكار المقسمة إلى أقسام مثل:
- أذكار الصباح
- أذكار المساء
- أذكار النوم
- تسابيح منوعة

كل ذكر له عداد يوضح عدد مرات التكرار المطلوبة، ويمكن للمستخدم تحديثه وتسجيل تقدمه.


## هيكل المشروع الرئيسي

- lib/main.dart: نقطة البداية للتطبيق
- lib/features/home: الصفحات والوحدات الخاصة بالصفحة الرئيسية
- lib/core: الكود المشترك مثل قاعدة البيانات والأنماط
- assets/data: ملفات البيانات الخاصة بالأذكار

## كيف تبدأ

1. شغّل المشروع باستخدام:
   flutter run
2. ابدأ من صفحة HomePage لفهم تدفق التطبيق
3. راجع ملفات الصفحات داخل lib/features/home/pages
4. ثم انتقل إلى قاعدة البيانات والبيانات داخل lib/core
